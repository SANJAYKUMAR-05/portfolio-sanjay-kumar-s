# Sanjay Kumar Portfolio

Modern full stack developer portfolio built with React.js and Vite.

## Tech Stack

* React.js
* Vite
* JavaScript
* CSS
* Framer Motion

## Features

* Responsive design
* Smooth animations
* Modern UI
* Projects showcase
* Skills section
* Contact section

## Run Locally

```bash
git clone https://github.com/SANJAYKUMAR-05/portfolio-sanjay-kumar-s.git

cd portfolio-sanjay-kumar-s

npm install

npm run dev
```
